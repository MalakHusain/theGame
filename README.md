# Memory Cards Game
## About the Game
This is a simple card-matching game where you have to find all the matching pairs under a 60-second time limit. I chose this project to practice using timers, tracking clicks, and shuffling lists in JavaScript.

## Getting Started
* **Play the Game:** 

### How to Play:
1. Click **Start** to get a 5-second sneak peek at the card locations.
2. Once they hide, click cards to flip them and look for matches.
3. Find all 6 pairs before the clock hits 0 to win.
4. Click **Reset** anytime to scramble the board and start a new game.
